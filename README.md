# Search Engine

Проект "Search Engine" - веб-приложение для индексации и обработки данных с различных веб-сайтов.

## Основные функции

- Индексация сайтов
- Обработка страниц
- Анализ содержимого

## Установка

1. Клонируйте репозиторий:

    ```bash
    git clone <repository_url>
    ```

2. Перейдите в каталог проекта:

    ```bash
    cd search-engine
    ```

3. Запустите приложение:

    ```bash
    ./run.sh
    ```
